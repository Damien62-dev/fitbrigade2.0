About FitBrigade


The Concept

FitBrigade is a web application designed to help fitness enthusiasts effectively plan and track their workouts. Focused on complete customization, it ensures you create a workout that's perfect for you. By creating your first workout, you join the fitness brigade; aka the "FitBrigade".


What's New in v2.0

  • Goals Feature: Set and track strength goals for specific exercises
  • Target Tracking: Define weight and rep targets with optional deadlines
  • Database Upgrade: Migrated from JSON to PostgreSQL for better performance

Why This App?

Passionate about running, and more recently weight training, I understand the complexity of creating a varied training program and tracking progress. I wanted to create a tool that simplifies and optimizes workout planning, while remaining 100% customizable. This application helps you visualize the muscle groups you're working and maintain a balanced training program.


Technologies Used

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Flask (Python)
  • Database: PostgreSQL with SQLAlchemy ORM
  • Hosting: Render.com
  • Version Control: Git & GitHub
